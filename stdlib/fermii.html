fdi12

DOCUMENT fdi12(x, b)
  return incomplete Fermi-Dirac integral of order 1/2,
     fdi12(x, b) = integral[b to inf]{ dt * t^0.5 / (exp(t-x)+1) }
  default accuracy to about 1e-10

SEE ALSO: fdim12, fdi32, fdi52
